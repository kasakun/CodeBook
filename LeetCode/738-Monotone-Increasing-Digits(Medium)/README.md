# Monotone Increasing Increasing Digits

Given a non-negative integer N, find the largest number that is less than or
equal to N with monotone increasing digits.

Example 1:

```bash
Input: N = 10
Output: 9
```

Example 2:

```bash
Input: N = 1234
Output: 1234
```

Example 3:

```bash
Input: N = 332
Output: 299
```
